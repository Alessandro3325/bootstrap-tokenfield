bootstrap-tokenfield
====================

A jQuery tag/token input plugin for Twitter's Bootstrap, by the guys from Sliptree

### Usage
	
	$('input').tokenfield()

### Features

* Copy & paste tokens
* Keyboard navigation, delete tokens with keyboard
